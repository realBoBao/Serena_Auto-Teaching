🧠 My AI Brain — API Documentation

REST API v1.0 — 25+ endpoints | Base URL: http://localhost:3005

🔐 Authentication: Most endpoints require Authorization: Bearer <REST_API_KEY> header.
Public endpoints are marked with PUBLIC

📊 System

GET/api/health PUBLIC

Health check — returns uptime, version, timestamp.

{"status":"ok","uptime":1234,"timestamp":"2026-06-04T00:00:00Z","version":"1.0.0"}
GET/api/cache/stats

Embedding cache statistics (hits, entries, hit rate).

💬 Q&A / RAG

POST/api/ask

Ask AI a question. Uses hybrid search (vector + BM25) + knowledge graph + self-reflect gate.

ParamTypeDescription
querystringQuestion text
{"ok":true,"answer":"...","source":"local"}

📚 Flashcards

GET/api/flashcards/due

Get flashcards due for review today.

GET/api/flashcards

Get random flashcards. Query: ?category=general&limit=20

GET/api/flashcards/stats

Flashcard statistics (total, due, by category).

POST/api/flashcards

Create a new flashcard.

{"question":"What is QuickSort?","answer":"A divide-and-conquer sorting algorithm...","category":"algorithms"}
POST/api/flashcards/:id/review

Review a flashcard (spaced repetition).

{"correct":true}
DELETE/api/flashcards/:id

Delete a flashcard.

🖼️ Vision

POST/api/vision/analyze

Analyze an image via Gemini Vision. Supports JSON (base64) or binary upload.

{"image":"base64...","mimeType":"image/png","prompt":"Describe this image"}

🏖️ Sandbox

POST/api/sandbox/run

Execute code in sandboxed environment. Supports Python, JS, C, C++, Java, Rust, Go, C#.

{"code":"print('Hello')","language":"python"}
GET/api/sandbox/languages

List supported programming languages.

⚔️ Debate

POST/api/debate

Run multi-agent debate. quick=true for 1-round, default is 3-round + sandbox.

{"problem":"Compare QuickSort vs MergeSort","quick":false}

🕸️ Knowledge Graph

GET/api/graph/search?q=entity

Search entities in the knowledge graph.

GET/api/graph/stats

Knowledge graph statistics (entities, edges, types).

GET/api/graph/export

Export graph as D3.js-compatible JSON (nodes + edges).

🧬 Self-Evolution

GET/api/evolution/stats

Self-evolution statistics (evaluations, model performance, A/B tests).

GET/api/evolution/gaps

Detected knowledge gaps.

GET/api/evolution/models

Model performance report (success rate, latency, cost per model).

📹 Videos

GET/api/videos

List rendered Manim videos.

POST/api/videos/cleanup

Clean up old videos (default: older than 48h).

🔔 Webhooks

POST/api/webhook/alerts PUBLIC

Receive alerts from Prometheus Alertmanager.

📝 Notes

POST/api/notes

Save a quick note (for iOS Shortcuts integration).

{"content":"Remember to review graph algorithms","tags":["study"]}
